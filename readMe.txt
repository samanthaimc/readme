Employee Management system configuration
-----------------------------------
Use following instruction to setup the project.

This application design Design REST API using spring boot to 
store, update, delete and retrieve employee data by using REST API.

the Project consist of two components;
1. backend 
2. frontend (Design by using angular)

The backend source needs to be upload a java server (jbosh).
the backend database is auto generated.

Frontend view as web page through the internet.

when setting up the above system on local environment,
the backend component needs to configure as a local server.
